## Project Idea and Purpose
I want to create a simple contact manager that would keep track of people and organizations I work with during my internship. The name of this project is JAWG Contacts and it would have a structured list of contacts that the Python program can read, be able to update and display information. Important details like the person’s name, organization, role, email address, and a short description of how we are connected to each other. The data would be saved in a simple format like a list of dictionaries, in a text format, CSV, or JSON file. 

This project would allow me to add a new partners, see all partners, and look up a specific person, organization, editing an existing contact, deleting a contact that is no longer active, or filtering by type of partner (like, “organizational name”, “provider”, interviewee” etc).

## Planned Features
Add a new partner with contact details
See a list of all partners
Look up a specific partner by name or organization
Edit an existing contract
Delect a contact that is no longer existing
Filter partners by type (organization, provider, interviewee)
